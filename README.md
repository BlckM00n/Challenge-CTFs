# 📚 Contenido

Este repositorio contiene distintos desafíos CTF enfocados en:

* Web Exploitation
* Privilege Escalation
* Reverse Engineering
* Criptografía
* Forense
* Enumeración
* OSINT
* Linux Exploitation

---

# 🧠 Objetivo

El objetivo de este repositorio es:

* Practicar técnicas de ciberseguridad en entornos controlados.
* Documentar soluciones y metodologías utilizadas.
* Mejorar habilidades de análisis y explotación.
* Mantener un registro organizado de desafíos resueltos.
* Utilizar GitHub como respaldo y portfolio técnico.
